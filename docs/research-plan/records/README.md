# Research Plan Records

This directory stores runtime research-plan outputs.

Each record should use `docs/research-plan/templates/research-plan-record.md`
and include status, request, triage, evidence inspected, decision, implementation
plan, testing strategy, risks, open questions, and recommended next action.

Template exports should keep this README as the directory contract. Runtime
records written here should not be exported back into the reusable template.
